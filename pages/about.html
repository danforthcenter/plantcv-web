About

PlantCV is an imaging processing package specific for plants that is built upon open-source software platforms OpenCV 1, NumPy 2, and MatPlotLib 3. PlantCV was created at the Donald Danforth Plant Science Center in 2014. Please check back for new functions and tutorials. Updates are available via twitter: Follow @plantcv Please help us attend to your questions, bug reports, and requests by posting them on our GitHub issues page.

If you use PlantCV, please cite the initial published description. See also below.
