Download instructions

Follow the link to a specific dataset for further information about the specific ways the dataset is available.

Datasets available through HTTP (including Figshare, Zenodo, Dryad, etc.) can be downloaded directly using a web browser or with a command-line download utility such as wget or curl.

Datasets available through CyVerse can be accessed through the methods described above for HTTP through the CyVerse Data Commons or via the CyVerse Data Store (account required) via the path /iplant/home/shared/danforth_center/.

Datasets available through Globus Connect can be accessed by signing up for a Globus account and accessing the Danforth Center Bioinformatics Endpoint following the instructions at https://bioinformatics.readthedocs.io/en/latest/globus/.
